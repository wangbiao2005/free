# Free - 终极工具箱

一个功能强大的工具箱应用，集成了多种实用工具。

## 功能特性

- 🎵 音频处理工具（转换、编辑、分析等）
- 🖼️ 图像处理工具
- 📝 文本处理工具（加密、OCR、PDF等）
- 🎬 视频处理工具（处理、字幕等）
- 💻 开发者工具
- 🏠 生活常用工具

## 技术栈

- React 19
- TypeScript
- Vite
- TailwindCSS
- Google Gemini API

## 快速开始

### 前提条件

- Node.js (推荐 v16 或更高版本)

### 安装步骤

1. 克隆仓库:
   ```bash
   git clone https://github.com/wangbiao2005/free.git
   cd free
   ```

2. 安装依赖:
   ```bash
   npm install
   ```

3. 配置环境变量:
   - 复制 `.env.local` 文件并重命名为 `.env`
   - 在 `.env` 文件中设置你的 `GEMINI_API_KEY`

4. 启动开发服务器:
   ```bash
   npm run dev
   ```

5. 构建生产版本:
   ```bash
   npm run build
   ```
